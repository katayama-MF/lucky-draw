# リール用画像 デザイナー発注仕様

抽選リール・当選表示・拡大表示で使う景品画像のサイズ・形式の指定です。

---

## 必須仕様（共通）

| 項目 | 指定 |
|------|------|
| **縦横比** | **1:1（正方形）** 必須 |
| **理由** | リール・当選表示で `object-fit: cover` により中央で正方形に切り抜いて表示するため。横長・縦長だと左右または上下が欠けます。 |
| **重要部分の配置** | **中央に配置**すること。端は画面によっては見えません。 |

---

## 推奨サイズ

| 用途 | 画面での表示サイズ | 推奨納品サイズ | 備考 |
|------|-------------------|----------------|------|
| リールのカード | 130×130px | **512×512px** または **1024×1024px** | 高解像度・大画面でも荒れないよう余裕を持たせる |
| 当選オーバーレイ | 最大 280×280px | 上記と同じで可 | 同一画像を流用 |
| タップで拡大 | 最大 約500×450px | 上記と同じで可 | 拡大時も同一画像を使用 |

**まとめ**: **512×512px** で納品すれば、現行の全表示で十分な画質です。  
さらに余裕を持たせる場合は **1024×1024px**。  
ファイル容量を抑えたい場合は **256×256px** でも可（リール表示 130px の約2倍＝Retina 程度）。

---

## ファイル形式・色

| 項目 | 指定 |
|------|------|
| **形式** | **PNG**（透過が必要な場合） または **JPG**（写真・イラストで透過不要な場合） |
| **カラーモード** | **sRGB**（Web表示のため） |
| **ファイル名** | 任意（アプリ側でスプレッドシートのURLや No. で管理するため、連番や賞品名など分かりやすい命名を推奨） |

---

## 表示仕様（参考・デザイナー向け）

- **リール上**: 130×130px の正方形で表示。角丸 12px。中央でトリミング。
- **当選時**: 最大 280×280px の正方形。角丸 24px。中央でトリミング。
- **タップ拡大**: 最大幅 500px 程度まで拡大。このときは `contain` のため画像全体が収まる（余白が出る可能性あり）。

上記のため、**構図は中央にメインを置いた正方形**で制作してもらうと安全です。
